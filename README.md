# Backstage POC Platform Service

This repository demonstrates Backstage capabilities:

- Catalog integration
- TechDocs documentation
- ADR governance
- API modeling
- CI quality gate
- Dependabot automation

## How to Validate

1. Push to GitHub
2. Ensure catalog discovery is enabled
3. Open Backstage
4. Verify:
   - Service appears in catalog
   - Docs render correctly
   - API entity is visible
   - CI status is displayed
   - Dependabot PRs appear
